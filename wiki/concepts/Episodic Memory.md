# Episodic Memory

One of three divisions of long-term memory in [[Lilian Weng]]'s cognitive-science framing of [[Agent Memory]]. Stores **specific past events** — time-indexed, context-rich observations.

> Example: "On Tuesday, the PostgreSQL cluster went down."

## Role in Agents

- Lets an agent recall *what actually happened* in a prior interaction, rather than just *what is generally true*.
- Provides the raw material for [[Memory Consolidation]] — repeated episodes distill into reusable [[Semantic Memory]] or [[Procedural Memory]].
- Episodic recall without consolidation causes agents to replay events without learning from them.
