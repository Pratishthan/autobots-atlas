# Procedural Memory

One of three divisions of long-term memory in [[Lilian Weng]]'s cognitive-science framing of [[Agent Memory]]. Stores **skills and workflows** — how to do things, not just what is true.

> Example: "When a user asks for a refund, first check the purchase date."

## Role in Agents

- Encodes repeatable procedures the agent can invoke without re-deriving them.
- Often emerges from distilling many [[Episodic Memory]] traces via [[Memory Consolidation]].
- Distinct from [[Semantic Memory]] (facts) — procedural knowledge is *action-shaped*.
- Overlaps in practice with the skills, tools, and playbooks catalogued in an [[Agent Harness]].
