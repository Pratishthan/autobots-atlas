# Vivek Trivedy

[[LangChain]] team member.

## Key Contribution

Coined the canonical formula for the [[Agent Harness]] concept:

> **"If you're not the model, you're the harness."**

This captures the binary distinction: everything in an AI agent system is either the model (weights, inference) or the harness (everything else). Tools, memory, orchestration, context management, state persistence — all harness.
