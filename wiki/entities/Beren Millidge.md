# Beren Millidge

Researcher; author of the 2023 essay "Scaffolded LLMs as Natural Language Computers."

## Key Ideas

In his 2023 essay, Millidge made the computer architecture analogy for [[Agent Harness]] design precise:

| Harness Component | Computer Analogue |
|---|---|
| Context window | RAM (fast, limited) |
| External databases | Disk storage (large, slow) |
| Tool integrations | Device drivers |
| The harness itself | Operating system |

His conclusion: **"We have reinvented the Von Neumann architecture"** — because it's a natural abstraction for any computing system that needs to store, retrieve, and act on information.

This framing is foundational to understanding why the [[Agent Harness]] is not incidental complexity but a necessary architectural layer.
